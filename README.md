# Glyphly

**Turn anything on your screen into language learning.**

Glyphly captures text from any app — games, video subtitles, PDFs, comics — reads it with on-device OCR, and helps you actually understand and remember it. Look up words the instant you see them, build a personal lexicon as you go, and lock it in with spaced-repetition review.

OCR and word lookups run entirely on your device. Optional full-sentence translation uses a cloud service, and only when you turn it on.

**Windows · Mandarin Chinese** — more languages on the way.

## Download

Get the latest installer from the **[Releases page](../../releases/latest)**.

After installing, Glyphly downloads its language engine once — then the core capture-and-review loop works fully offline. It keeps itself up to date automatically.
